## 职业

现版本Wynncraft总共有5个职业，每个职业按照技能树左中右又分为三个特化流派

所有职业都有四种技能，虽然在技能树上位置不一定相同，但是同一个鼠标按键序列对应技能，往往功能上是相近的

| 技能  | 序列  | 功能     |
|-----|-----|--------|
| 一技能 | RLR | 次要输出技能 |
| 二技能 | RRR | 位移技能   |
| 三技能 | RLL | 主要输出技能 |
| 四技能 | RRL | 流派特化技能 |

当然这只是粗略的分类，并不是所有技能树都符合这个特点
