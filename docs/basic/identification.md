# 词条系统

Wynn中的物品(武器，盔甲，饰品)往往都有**词条**

上面是两把水神话弓Spring的词条

对于同一个物品，词条的列表是固定的，但是鉴定度往往不同。这也就是说，所有的Spring都有且仅有这些词条，但是词条具体给多少值，则取决于其鉴定度

物品的非技能点词条往往有一个**鉴定范围**。鉴定时，会随机在范围内选取一个值作为该词条的鉴定度

:::tip
你可以在Wynncraft官网、WynnBuilder或者直接在wynntils的物品指南中查询物品词条的鉴定范围，也可以直接在游戏中将鼠标悬停在物品上按下shift来查看
:::
