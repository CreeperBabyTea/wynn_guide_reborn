## 问前三思

- 我真的不能靠自己解决这个问题吗？我是否尝试了所有可能的办法？
- 我这个时间在这里提问合适吗？
- 我是否尝试过重新做一遍，检查有没有遗漏？

  然后你应该搜索互联网上是否已经有相关资源。请你知道STFW和RTFM是什么意思

  >STFW: Search the f**king web

  >RTFM: Read the f**king manual

  ## 提问时

  首先你必须要思考你提供的信息是否足够

  请不要问：

  为什么我的珍珠炮用不了？

  而是问：

  我装了某某某的珍珠炮，视频链接啥啥啥，为啥开火的时候我会被传送到岩浆池子里？

  其次，请放低身份

  群友不是你爹妈，群友没有义务解答你的问题

  如果你的态度不端正，那群友完全可以拒绝回答，同时把你转发到别的群，成为笑料

  我们不要求你卑躬屈膝叠一大堆谦辞敬辞，但是你应该保持基本的礼仪，绝大多数群友是很愿意帮助你的（前提是你遵守了提问前的三条规则）

  ## 最后，如果你觉得这个群里你完全得不到答案，请退群

  如果是群友真的有病，你留下也没用

  如果是的问题，群友不会想要你留下
