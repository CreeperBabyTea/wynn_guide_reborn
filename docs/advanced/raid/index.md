Raid(袭击)是一种特殊的多人高难副本，你可以在Raid中获取到大量资源

且对于完整游戏体验来说，Raid是必要的游戏内容。

在这里，我会对Raid进行详尽的讲解，包括如何解锁各个Raid，以及各个Raid的机制、奖励等。

:::danger

非常不建议在不了解机制的情况下匹配路人打Raid

因为部分Raid机制会因为一个人的失误导致全队失败

>对就是tna的树洞

:::

:::tip

低等级的Raid存在根据玩家等级浮动的动态难度

如，使用70级的账号进入Notg，该玩家视角下的敌人数值会比满级玩家低得多

同时Raid结算界面的伤害也是根据当前Raid的等级下限而定的

:::

现版本wynncraft有四个raid：Nest of the Grootslang(notg)，Orphion's Nexus of Light(nol)，The Canyon Colossus(tcc)，The Nameless Anomaly(tna)

## 流程

首先，玩家需要通过RaidKeeper，或partyfinder(/pf)加入Raid队列

进入队列时，需在身上携带对应的门票(runes)，匹配成功后自动扣除

一场raid有四个关卡：三个挑战关，以及最后的boss关。特别的，nol的boss分为两关

每个关卡前会有一个准备房间，会有文字介绍关卡的任务或带你去boss房；关卡房间和下一个关卡的准备房间之间还有休息房间，这里你可以选择一个buff以及在blacksmith处修复你的装备

一般而言，挑战关卡大致内容分为如下几类：
- 守点，需要一名玩家站在平台上启动倒计时，其他玩家清理周围定期刷新的小怪和miniboss保护平台和守点玩家
- 收集，房间分为若干分支任务，包括战斗、跑酷、解密等，完成任务获得材料，直到收集足够数量的材料
- 战斗，击杀指定的怪物或miniboss

:::tip

正常通过一个挑战关，会获得3 Reward pulls + 1 Aspect pull

在挑战房中死亡，会导致最终奖励数量减少，仅获得 2 Reward pulls，无额外Aspect pull

:::

:::danger

在Raid期间掉线了，可以直接重新连接并回到raid当中。但是如果这发生在挑战房间，你重进后是死亡状态

在休息处掉线基本可以稳定重连，但如果队友已经进入下一个挑战房间，依然会判定你已在该挑战中死亡

:::

## Gambit

在Raid准备界面，您可以选择启用Gambit来**增加难度**以获得额外奖励

每天凌晨(对应美国冬令时/夏令时的中午十二点)，系统会从所有Gambit中随机抽取**四个**作为当日Gambit

在满级时，最多可以开启**三个**Gambit，并且开启的第1/2/3个gamibt分别会为您提供额外的1/3/5战利品奖励和1/2/3 aspects奖励

:::warning

有些gambit会极大影响部分职业，甚至导致完全玩不了，不要为了多选gambit而放弃游玩能力

>75%resis我还玩个几把bolt

:::

## Guild Raid

Guild raid简称gr，是由工会成员进行的高强度raid挑战

当队伍中的玩家全部来自于同一个工会时，Raid准备界面会有所变化。鼠标悬浮于右侧牛头上可以看到guild raid选项

如果四名玩家均选择Opt in，则会启动**Guild Raid**，所有敌人的数值均有所提升，且BOSS会有额外机制

>gtcc pog

如果队伍成功完成了Guild Raid，不仅此次战斗的奖励会有所提升，还会额外获得公会资金/赛季点/公会经验等诸多奖励
