## 什么是配装？

配装就是一整套装备，同时往往也带一套技能树

一套合格的配装往往能提供输出能力或辅助能力，并且具有一定生存能力

配装的第一要求肯定是能穿上配装、能拿动武器，也就是技能点要足够

一个100级的存档拥有200个可分配技能点，同时任意属性你只能分配最多100点，因此高于100就需要装备来提供点数

除此之外，技能点的增益上限是150点，超出150的点数是无效的~~对吧dz~~

### 元素

配装往往强调两种或两种以上的元素装备组合。最常见的配装往往采用三元素组合或彩虹。每种元素具有其特点：

- 地E(str)：较高的血量，较高的伤害，强化近战
- 电T(dex)：较低血量，极高伤害，依靠ms续航
- 水W(int)：较高技能伤害，依靠mr续航
- 火F(def)：肉，高回血
- 气A(agi)：高移速，技能减cost，脆

:::warning
这些都只是一般规律，并非所有装备都符合
:::

常见元素组合：

- 彩虹：最全能的组合，样样都会但往往样样不精
- ETF：新手推荐组合，有肉有输出，但是Spell流可能回转比较艰难
- ETW：玻璃大炮，高风险但是高伤害
- TWA：性能比较均衡，但是缺少地点数使伤害受限
- EWA：和TWA一样比较均衡，伤害稳定

### ehp

ehp即effective health，火属性和气属性都会影响ehp。然而，由于气属性的闪避随机性很大，我们更喜欢参考ehp(no agi)，即忽略闪避的ehp

>使用agi享受随机被秒杀

ehp随玩法的变化差别也很大，有些玻璃大炮可能只有7k甚至5k ehp，而作为坦克的圣骑士则可以轻松达到100k以上

## 战斗

### 输出

大部分配装都需要考虑输出的问题。输出词条主要包含三种：技能点、百分比伤害、原始伤害

技能点方面，最重要的是str和dex。除此之外各元素对应的技能点也有一定伤害加成，如int点数对水伤有加成

然后两种词条伤害，粗略的理解，百分比伤害是乘在在武器基础伤害(base dps)之上，而原始伤害则是直接相加，且原始伤害不受百分比伤害影响

因此，比如我们有一把base dps为594的武器，如果在10%spell和72raw spell中选择，72raw spell的收益就更高

>什么武器才594base啊，hero说到

:::tip
一般而言，武器基础面板为：

弓手>萨满、刺客>战士>法师

因此百分比对弓手、萨满、刺客有更好效果，原始伤害对法师有更好效果，而战士则需要权衡
:::

### 回转

回转一般包括两种，蓝量回转和血量回转。血量回转是指依靠烧血的fallen和依靠血池的acolyte，在战斗中维持自己的血量的能力。而蓝量回转则是普通的烧蓝的职业

对于消耗蓝量的职业，需要通过mr或ms或技能树技能维持蓝量：

- Spell Spam：直接依靠高额mr回转。常见于带有水属性的配装
- Spell Steal：将攻速压低到super slow，在技能间穿插平a偷蓝。常见于电属性配装
- 双回转：上述两种都有

部分职业有特别的回蓝技能，如bolt的Recycling和奥法的奥术转换。不同职业的蓝耗也往往不同，如血萨的蓝耗就远远低于bolt的

## 基本配装思路

1. 选择一把武器，选择一个职业分支。有些职业分支有特别适配的武器，比如萨满的oly适配召唤师，法师的lament适配lb。具体而言可以参考神话介绍，或问问群友
2. 买不起神话的话，有一些神话有下位取代，一样可以参考神话介绍中的内容。除此之外也可以在wynnatlas查询，网址见[新手教程](/tutorial/)
3. 确定元素组合。新手比较推荐使用ETF、EWF等，最好E/T带一个，F/A带一个(优先F)，再加上W提供回转
4. 饰品的选择上，手镯无脑Prowess，项链选择li岛对应元素的，戒指选择li戒指或者summa提供点数和回转
5. 装备的选择上，最重要的是能穿上，其次考虑回转和输出。点数不够且对输出要求不太高的情况下裤子可以使用Vaward。hive的装备往往比较强力，可以优先使用

:::tip
万能公式

头盔：Spectrum/Sequencer
胸甲：Libra
裤子：Aleph Null/Vaward
鞋子：Epilogue/Capricorn
戒指：Summa/Moon Pool Circlet/diamond li
手镯：Prowess
项链：Gigabyte/Collection/diamond li
:::

:::danger
关于hive装备 配装时务必时刻记住，hive同层的装备是无法同时装备的，否则会受到大量负面效果

例如，Prowess与Contrast，Intensity这三件饰品只能同时装备一件在身上
:::

### 升级配装

在达到一定水平并积累一定绿宝石后，一开始的装备一般都需要升级

forum上的[这篇文章](https://forums.wynncraft.com/threads/the-ultimate-build-guide.320092/)包含了非常优质的配装设计，可以直接抄作业

如果你想自己搞定的话，基本步骤如下

:::warning
由于毕业配装往往价格不菲，因此请先在Wynnbuilder中设计好再进行采购
:::

第一步考虑一双合适的神话鞋。一般而言就是Stardew或Crusade Sabatons。除此之外，部分特殊的武器也会用到其他神话鞋，可以参考神话一览中的介绍

第二步，利用Wynnatlas检索你所需要的装备。你可以点击右上角进入高级搜索模式，并且根据你的武器调整表达式。

比如，当你需要查找Fatal的头盔时，你可以在Filtered by中填

`type="helmet"`

在Sorted by中填

`5.983*(spelldam%+thunderdam%+elementaldam%)+spellrawdam+thunderdamraw+elementaldamraw`

同时也需要考虑点数问题，有些装备虽然词条给的伤害不算特别多，但是提供了超高点数加成(如Bete Noire，Moontower等)

第三步，由于装备强度往往是戒指<手镯、项链<除鞋以外的盔甲<鞋，所以我们需要把最弱的戒指替换掉。戒指一般可以使用自制(crafted)戒指，配方可以找工会要/找群友要/扒已有配装的

最后，如果点数不够，则对每件装备进行调整取舍，直到能够穿上

## 特殊配装

### 近战流

近战流分两类：Tier Stack(简称ts)和Heavy Melee(简称hm)(还有已经死掉的panic zealot cancel stack)

ts的思路是选择攻速为Super Slow的武器，并且叠堆攻速词条直到Super Fast，以获得高额近战伤害

hm的思路是利用攻速保底，选择Super Slow的武器，并且装备降低攻速、提供超高近战伤害的装备，通过地粉技能打出超高伤害。

一般而言，近战流只有在tcc有发挥空间，因为tcc的buff有相当强力的近战伤害增益

### Fallen

腐化战士依靠血量释放技能，因此对蓝量回转没有特别需求。腐化战士分为全烧和半烧，全烧即完全使用血量输出，半烧则有少量的蓝量回转

对于全烧的fallen，可以使用一些极大降低蓝量回转但是提供高额伤害的装备(如Caesura)

曾经有一个扣除大量mr但是提供42spell%的腐化专用自制戒指，但是如今被削入土只剩28spell%

但是，即使是全烧fallen也需要一定的int和减cost，以减少每个技能的消耗，延长续航
